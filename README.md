# Agro Manager Startup

Base Expo inicial usando `App.tsx` e `React Navigation` stack.

## Rodar o projeto

```bash
npm install
npm start
```

## Estrutura atual

- `App.tsx`: ponto de entrada da aplicação
- `components/`: componentes compartilhados e domínios em preparação
- `assets/`: imagens e arquivos estáticos
